# Prompt++

An AI-powered prompt optimization tool built using FastAPI, JavaScript and Ollama.

## Features

- Prompt optimization
- 10 optimization categories
- Local AI (Ollama)
- FastAPI backend
- Responsive UI

## Tech Stack

- HTML
- CSS
- JavaScript
- FastAPI
- Ollama
- Qwen2.5

## Run

Backend

```bash
uvicorn app.main:app --reload
```

Frontend

Open with Live Server.
